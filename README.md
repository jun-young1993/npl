# Usage
```shell
$ nf --help

[Usage]
   npm-fuse 1.0.0

   $ npm-fuse <option>
   $ nf <option>

[Options]
   --create                      create new project 
                                 $ nf --create
   --config                      get npm-fuse config list
                                 $ nf --config
   --config --set <key=value>    update npm-fuse config list
                                 $ nf --config --set path=/myProjects/source

```
